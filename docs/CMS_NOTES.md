# MansHockey Cloud CMS

Den här versionen lägger till CRUD mot D1:

- GET hämtar poster
- POST skapar poster
- PUT uppdaterar poster
- DELETE tar bort poster

Resurser:
- /api/matches
- /api/scout
- /api/media
- /api/travel
- /api/documents

Adminläge aktiveras i UI. Nästa säkerhetssteg är Cloudflare Access.
